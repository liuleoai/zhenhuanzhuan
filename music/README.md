# 音乐文件说明

此文件夹用于存放游戏背景音乐文件。

## 需要的文件：

1. **bgm.mp3** - 游戏背景音乐
   - 风格：古风、优雅、舒缓
   - 用途：游戏进行时的背景音乐
   - 推荐乐器：古筝、笛子、二胡等
   - 音量：30%
   - 循环播放

2. **tense-bgm.mp3** - 紧张情节音乐
   - 风格：古风、紧张、戏剧性
   - 用途：重要情节和结局时的背景音乐
   - 推荐乐器：鼓点、弦乐等
   - 音量：40%
   - 循环播放

## 如何添加音乐：

1. 下载或录制合适的古风音乐文件
2. 将文件重命名为对应的名称（bgm.mp3 或 tense-bgm.mp3）
3. 确保文件格式为 MP3
4. 将文件放入此文件夹

## 音乐推荐：

- 搜索关键词："古风音乐"、"宫廷音乐"、"中国风音乐"
- 免费音乐网站：Free Music Archive、Incompetech
- 确保使用免费或已获得授权的音乐

## 注意事项：

- 音乐文件大小建议不超过 10MB
- 音乐时长建议 3-5 分钟，适合循环播放
- 如无音乐文件，游戏仍可正常游玩，只是没有背景音效
